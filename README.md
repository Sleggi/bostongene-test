### Тестовое задание для bostongene без стилей

#### technologies, libraries used in this project - html, css(flexbox), react, Formik, yup, NumberFormat, uuid
##### functionality: 
CRUD приложение, которое представляет собой форму отправки
объявления и содержит следующие поля:
- Заголовок: не более 140 символов (обязательное поле);
- Текст объявления: не более 300 символов (необязательное поле);
- Телефон: номер в формате +7 (ххх) ххх-хх-хх (обязательное поле);
- Выбор города из списка: Москва, Санкт-Петербург, Казань, Нижний Новгород
(необязательное поле);
Есть возможность создавать объявления через форму и видеть
их список (от новых к более старым). У созданных объявлений есть возможность
удаления.
